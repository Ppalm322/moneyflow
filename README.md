# MoneyFlow

แอปบัญชีรายรับรายจ่ายส่วนตัว (เว็บ) — เก็บข้อมูลในเครื่องด้วย `localStorage` ใช้งานออฟไลน์ได้ ไม่มี backend

## โครงสร้าง

```
index.html        หน้าเว็บแอป (sidebar + แดชบอร์ด + ฟอร์ม + รายการ)
style.css         layout ของหน้าเว็บ
shared/core.js    data layer + business logic + ตัวสร้าง UI + โมดัล/toast
shared/base.css   design tokens + คอมโพเนนต์ (การ์ด/กราฟ/ฟอร์ม/รายการ/โมดัล)
```

## รันในเครื่อง

ต้องเปิดผ่าน HTTP server (โหลดไฟล์ `shared/*` แบบ relative):

```bash
cd moneyflow
python3 -m http.server 8755
# เปิด http://localhost:8755/
```

## ฟีเจอร์

- บันทึกรายรับ/รายจ่าย พร้อมหมวดหมู่ + แนบรูปสลิป (บีบอัดอัตโนมัติ)
- การ์ดสรุปยอดวันนี้/สะสม + กราฟวงแหวนสัดส่วนค่าใช้จ่าย (วันนี้/ทั้งหมด)
- ค้นหา + ตัวกรอง (วันนี้ / 7 วัน / เดือนนี้ / รายรับ / รายจ่าย)
- แก้ไข/ลบรายการ, ลบรายการวันนี้, ส่งออก CSV, ดูสลิปแบบ lightbox
- เปลี่ยนวันอัตโนมัติ (ยอดรายวันเริ่มนับใหม่ ประวัติ/ยอดสะสมคงเดิม)

ข้อมูลเก็บในเบราว์เซอร์ — พร้อมต่อ API ภายหลังผ่าน Data Layer (`Store`) ใน `core.js`
